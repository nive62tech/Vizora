# Phase 3 — React Frontend

## What this phase covers

- Vite + React app with Tailwind CSS dark theme
- File drop zone accepting CSV, Excel, JSON
- Sidebar showing file info after upload
- Chat window with message bubbles
- Placeholder AI responses (real AI in Phase 4)
- Axios service layer talking to FastAPI backend

## How to run

From D:\projects\Vizora\phase-3-frontend\client:

```bash
npm run dev
```

Open http://localhost:5173
