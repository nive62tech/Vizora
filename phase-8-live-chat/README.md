# Phase 8 — Live Dashboard Chat

## What this phase covers

- POST /dashboard/edit endpoint for live dashboard editing
- Live edit chat panel on the right side of dashboard view
- Supported edit commands:
  - "add chart 3" — adds chart from library to dashboard
  - "remove chart 1" — removes chart from dashboard
  - "make chart 2 a line chart" — changes chart type live
  - "rename this dashboard to Sales Report" — renames dashboard
- All changes persist to dashboards.json instantly
- Charts update without page reload

## How to use

Open any dashboard then use the Live Edit panel on the right.
