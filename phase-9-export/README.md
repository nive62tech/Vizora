# Phase 9 — Export

## What this phase covers

- PNG export on individual charts using Plotly.toImage
- HTML export on individual charts — standalone offline file
- HTML export on full dashboards — all charts in one file
- PDF export on dashboards using browser print dialog
- All exports happen 100% locally in the browser
- No server involved, no data sent anywhere

## How to use

### Chart export
Each chart in the chat has PNG, HTML, PDF buttons.

### Dashboard export
The dashboard header has PNG, HTML, PDF buttons.
- PNG — use individual chart PNG buttons
- HTML — downloads a standalone HTML file with all charts
- PDF — opens browser print dialog
