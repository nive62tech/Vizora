Features

Everything you need.
Nothing you don't.

Built for analysts, teams, and anyone who cares about their data staying private.

🤖
Local AI — No API keys
Ollama runs the AI model entirely on your CPU. No OpenAI, no Anthropic, no internet required after setup.
💬
Chat with your data
Ask questions in plain English. "Show me salary by city as a bar chart" — Vizora understands and delivers.
📊
Auto-numbered charts
Every chart gets a number. Build dashboards by saying "create a dashboard with charts 1, 2 and 4".
🎛️
Live dashboard editing
After building a dashboard, keep chatting to edit it. Add charts, remove charts, change types — live.
🌐
LAN team mode
One machine hosts, the whole team connects via browser on the same WiFi. Data never leaves the building.
📤
Export anywhere
Export charts and dashboards as PNG, HTML, or PDF. Offline HTML files work without any server.
How it works

Up and running
in minutes.

One script handles everything. No DevOps required.

1
Install the prerequisites
Download Python, Node.js, and Ollama. Takes about 5 minutes total. All free, all open source.
2
Run setup.bat
Double-click setup.bat. Pick your model size. Everything installs automatically — packages, dependencies, the AI model.
phase-12-installer\setup.bat
3
Drop your data file
Drag a CSV, Excel, or JSON file into the sidebar. Vizora reads it instantly and shows you the columns.
4
Chat and visualize
Ask questions, request charts, build dashboards, export reports — all in plain English, all locally.
